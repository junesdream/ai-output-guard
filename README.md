# AI-Output-Validator 🛡️

Automated validation suite for LLM prompt-response pairs. Designed for QA Engineers and AI Developers.

## 🚀 Overview
This tool helps to automate the "Human-in-the-loop" process by scanning LLM outputs for quality issues, safety filters, and length requirements.

## 🛠 Features
- **JSON Batch Processing:** Validates large datasets in seconds.
- **Safety Auditing:** Detects forbidden keywords or filter placeholders.
- **Reporting:** Generates detailed JSON reports for audit trails.

## 💻 Installation & Usage
1. Clone the repository.
2. Ensure you have `data.json` in the root folder.
3. Run the validator:
   ```bash
   python main.py
   ````
